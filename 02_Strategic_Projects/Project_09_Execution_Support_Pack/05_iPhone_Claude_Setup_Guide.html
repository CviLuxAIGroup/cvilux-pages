SETUP STEPS · 6 步完成

基本設定

15 分鐘可完成。所有功能皆 已驗證 之 Anthropic 官方支援項目。

1
STEP 1

下載 Claude for iOS
  1. 1. 打開 iPhone 上的 App Store
  2. 2. 搜尋 Claude（開發者：Anthropic, PBC）— 認準此開發者，避免下載到仿冒
  3. 3. 點選「取得」下載並安裝
  4. 4. 開啟 App，使用您現有 Anthropic 帳號登入（與電腦版同一帳號）
  5. 5. 選擇登入方式：建議用「Sign in with Google」或公司 Email — 與後續 Workspace 整合一致
📎 官方文件：support.claude.com/installing-claude-for-ios
2
STEP 2

登入瀚荃集團 Workspace

如集團已開通 Claude Team / Enterprise（依 Project_05 AI 轉型組企劃）：

  1. 1. 在 App 中切換至「Workspace」帳號（左上角頭像 → Switch Workspace）
  2. 2. 用集團 SSO 登入（如已配置 Azure AD / Entra ID）
  3. 3. 確認可看見集團共用之 Projects（P01-P09）與 Skills（cvilux-brand-cis、enterprise-ai-platform-website 等）
  4. 4. 個人 vs 公司分離：建議保留個人帳號處理私人事務，公司事務一律用 Workspace 帳號 — 對話歷史不混淆
⚠️ 資安原則：公司資料（CIS、客戶名單、報價）一律走 Workspace；Workspace 預設「對話不用於模型訓練」，符合企業隱私要求。
3
STEP 3 ★ 強烈建議

設定 Action Button + 鎖定畫面快捷

iPhone 15 Pro / 16 / 17 系列 有 Action Button（動作按鈕），可一鍵直接呼叫 Claude — 對董事長最有用！

  1. 1. 開啟 設定 → 動作按鈕
  2. 2. 滑到「捷徑」選項
  3. 3. 選擇「Claude」中的 New Conversation（開新對話）或 Voice Mode（語音模式）
  4. 4. 完成後，長按側面 Action Button 即可立即呼叫 Claude

鎖定畫面 / 控制中心：

  1. 1. 鎖定畫面長按 → 「自訂」 → 加入 Claude widget
  2. 2. 控制中心可加入 Claude 快捷（iOS 18+）
  3. 3. 不需解鎖手機即可開始對話（要確認權限）
📎 官方文件：Lock Screen, Control Center, Action Button
4
STEP 4

啟用 Voice Mode 語音對話

開車時、走路時、巡廠時直接「對 Claude 說話」，不用打字。

  1. 1. 在 App 主畫面，點下方 麥克風 / Voice 圖示
  2. 2. 第一次使用會請求麥克風權限 — 允許
  3. 3. 直接說話，Claude 會即時回答（含語音播放）
  4. 4. 語音對話結果會儲存於對話歷史，事後可閱讀文字

另有 Dictation 聽寫功能（不開語音模式時）：

  • 輸入欄位的麥克風圖示 → 邊說邊轉文字
  • 適合會議中私下記錄、想到什麼立即口述
📎 官方文件：Voice Mode · Dictation
5
STEP 5

設定 Siri 捷徑 + 主畫面 Widget

把您常用的問題包裝成一句話 Siri 命令，例如：

  • · 「Hey Siri, 開瀚荃 ESG 助手」 → 自動開 Claude + 預設提示「我是楊董，請以 ESG Agent 角色回答」
  • · 「Hey Siri, 翻譯英文簡報」 → 自動開語音模式
  • · 「Hey Siri, 查 CviLux 連接器規格」 → 預設搜尋 RAG

設定方式：

  1. 1. 設定 → Siri → App 捷徑
  2. 2. 找到 Claude → 啟用「捷徑」與「App Intents」
  3. 3. 在「捷徑」App 中可建立自訂流程（如 + Claude 啟動 + 預填指令）
  4. 4. 主畫面長按空白 → 加入 Widget → 找 Claude → 選大小
📎 官方文件：App Intents, Shortcuts and Widgets on iOS
6
STEP 6

啟用 Share / 分享給 Claude

收到的 PDF、Email、新聞、相片可直接分享給 Claude 分析：

  1. 1. 在任何 App（Safari / Mail / Files / Photos）點分享按鈕
  2. 2. 第一次使用點「更多」→ 加入「Claude」到分享清單
  3. 3. 之後可直接「分享 → Claude」 → Claude 開新對話並接收內容
  4. 4. 應用情境：收到客戶 RFQ → 分享給 Claude → 「分析重點 + 建議回覆」
📎 官方文件：Using Claude With iOS Apps
USE CASES · 5 個董事長使用情境

日常使用情境

配合您「董事長兼永續長」雙角色 — 主席 / 巡廠 / 出差 / IR 場景。

🏭

情境 1 · 巡廠檢視 IoT

在東莞 / 蘇州 / 越南廠走線時，看到 IoT 設備異常或想了解 OEE。

操作步驟
1. 拍照 / 錄影 → 分享給 Claude
2. 語音問：「這台設備可能是什麼異常？」
3. Claude 結合 sCMS Agent 給診斷與建議
🗣

情境 2 · 主持會議

董事會、客戶簡報、員工大會等大型會議。

操作步驟
1. 開啟 Voice Mode 在桌上
2. 會議結束說：「整理重點與行動項目」
3. Claude 給 5 條結論 + 待辦清單
✈️

情境 3 · 出差路上

機場 / 高鐵 / 計程車中，需要快速取得資料或回覆。

操作步驟
1. 長按 Action Button → Voice Mode
2. 「給我 CviCloud sCMS 商模 4 方案重點」
3. cvilux-brand-cis Skill 自動載入回答
🌱

情境 4 · ESG / CBAM 諮詢

永續長身份，需要快速回應 ESG / 法規問題。

操作步驟
1. Siri：「Hey Siri, 開 ESG 助手」
2. 預設導向 ESG Agent + RAG（碳定價辦法、CBAM）
3. 聽 / 看回答後可立即轉發給法務
📰

情境 5 · 媒體訪談 / IR

記者來電 / 法說會 / 投資人問話。

操作步驟
1. 訪前：分享記者來信給 Claude → 預備答覆
2. 訪中：偷偷 Dictation 輸入難題 → 即時建議用詞
3. 訪後：請 Claude 整理為新聞稿草稿
📅

情境 6 · 行程 / 任務管理

個人助理級別的支援。

操作步驟
1. 「整理我今天的行程」 + iCal 整合
2. 「明天會議的 3 個重點」 + Project 文件 RAG
3. 「記得提醒我下午 4 點...」

SECURITY · 安全與隱私

8 條董事長必知資安建議

★ 必做 1

Face ID + 鎖屏

確認 iPhone 鎖屏密碼複雜（≥ 6 位數字 + Face ID），避免遺失時資料外洩。

★ 必做 2

App 內 Face ID 鎖

設定 → Face ID → 加入 Claude → 每次開啟需 Face ID（避免他人借手機看到對話）。

★ 必做 3

公私分離帳號

公司事務一律 Workspace 帳號；個人用 Pro 帳號。雙帳號可在 App 內快速切換。

★ 必做 4

商業機密不口述

語音模式於公開場合（餐廳、會議室、車內）勿問商業機密問題（成本、客戶名）— 周邊人會聽到。

建議 5

大陸區使用

推論 Claude 服務在大陸需 VPN；建議過境前先在台灣查詢好需要的資料，或使用集團 SSL VPN 連回台灣存取。

建議 6

客戶名單不外傳

如要分析客戶資料，先用代號（A 公司、B 公司），避免完整客戶名 + 報價金額同時出現於對話。

建議 7

分享對話前審視

分享對話連結給其他人前，先檢視整段對話是否含敏感資料（NDA / 客戶 / 內部成本）。

建議 8

遺失手機應對

立即用其他裝置 Apple Find My 鎖機 + 遠端清除；同時通知 IT 撤銷 Workspace token。
