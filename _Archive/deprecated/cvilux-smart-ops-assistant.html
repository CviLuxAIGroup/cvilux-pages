鼎新 WF AIGP ERP + 聯詮 BI

  • 鼎新 AIGP ERP：含 AI 助手，九大循環作業單一入口，台灣總部主系統
  • 聯詮 BI（LCNet）：專為鼎新ERP開發的 BI 可視化，深度整合製造業報表
  • 九大循環原始 KPI 數據源：訂單、生產、採購、庫存、財務等全由鼎新輸出
  • 聯詮BI負責鼎新端的製造BI看板，含即時生產監控、良品率、OEE等
  • 聯詮BI可與戰情室 Embed 整合，減少重複開發
層次工具職責
生產層鼎新 WF AIGPERP 主系統、AI 助理、作業循環
分析層聯詮 BI製造KPI、生產良品、財務報表
介面層聯詮 Embed戰情室看板嵌入呈現

Microsoft 生態系 BI 整合

  • Power BI：MS生態系端的BI主工具，整合 D365、M365、Azure 數據
  • Microsoft Fabric：數據中台，整合鼎新、D365、CRM多源數據的中央倉儲
  • Power BI Embedded：可直接嵌入戰情室看板，無需另建前端
  • Azure Data Factory：ETL 工具，定期從鼎新拉取資料同步至 Fabric
  • Copilot for Power BI：自然語言查詢（「本月毛利最低的產品線？」）
層次工具職責

  • Google Workspace：高階幕僚協作（Docs、Slides、Meet、Drive）
  • Gemini for Workspace：AI 會議摘要、策略文件起草、多語翻譯
  • Vertex AI：集團視角的跨廠ESG分析、獲利預測、供應鏈AI
  • BigQuery：集團BI數據倉儲，整合台灣/海外/ESG全域數據

  • 大陸廠：鼎新→阿里雲中轉（脫敏）→GCP香港節點→BigQuery

戰情室看板 輕量整合建議

  • 方案A（推薦）：Power BI Embedded 為主看板，聯詮BI製造模組以 iFrame 嵌入，Google Looker Studio 幕僚看板獨立頁
